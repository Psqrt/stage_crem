0/ Veuillez trouver la documentation de l'application dans le répertoire ./extra/ (Documentation.Rmd).
1/ Veuillez vérifier l'installation des packages nécessaires en lançant le code 0_Packages.R dans ce répertoire.
2/ Le lancement de l'application se fait à partir du fichier 1_fichier_lanceur.R (voir documentation).
3/ Veuillez utiliser de préférence l'application dans un navigateur web pour le bon affichage des images et des éléments cartographiques.
4/ Veuillez consulter la section Bugs connus de la documentation.


L'application a été testée sur différents systèmes d'exploitation :
	- Ubuntu                  : Pas de problème identifié
	- Debian (salle A112-113) : Pas de problème identifié
	- macOS                   : Pas de problème identifié
	- Windows 7 (batiment B)  : Problème avec l'option stringsAsFactors de la fonction fread() (package data.table)
	- Windows 10              : Problème d'affichage du panel supérieur (inexistant) dans la cartographie.
